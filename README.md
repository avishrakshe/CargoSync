# CargoSync – Premium Fleet Flow Dashboard

A premium, visually distinctive Main Dashboard for CargoSync Fleet Management built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- **Animated Sidebar** – Slide-in on load, collapse/expand, active indicator with layout transition, hover effects
- **Dashboard Header** – Fleet Flow title, animated search bar, Group By / Filter / Sort dropdowns, New Trip & New Vehicle buttons
- **KPI Cards** – Active Fleet, Maintenance Alerts, Pending Cargo, Utilization Rate % with count-up animation, glassmorphism, 3D hover tilt
- **Fleet Data Table** – Trip ID, Vehicle, Driver, Status with color-coded badges, expandable rows, search, filter, sort, pagination
- **Visual Polish** – Dark futuristic theme, neon blue/purple accents, animated background gradient, notification bell, live indicator, floating action button

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
